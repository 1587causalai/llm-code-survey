# 快速开始

## 环境准备

1. 克隆项目
```bash
git clone https://github.com/yourusername/llm-code-survey.git
cd llm-code-survey
```

2. 安装依赖
```bash
# 安装文档工具
npm i docsify-cli -g

# 安装Python依赖
pip install -r requirements.txt
```

## 项目结构说明

- `paper/`: 研究论文及相关资料
- `docs/`: 项目文档和教程
- `research/`: 研究过程记录
- `src/`: 代码实现
